### Real-Time Directions Prototype

Solving the Max-Flow problem using the optimizing nuZ solver included in the Z3 theorem prover.


## Requirements
 
- Clang 3.6+ or GCC 4.9+
- Boost 1.55+
- Z3 4.4.1+


## Building

    make


## Usage

    ./MaxFlowSolver
